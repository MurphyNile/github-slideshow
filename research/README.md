# Research

Key ideas that run through Murphy Nile's projects.

Murphy Nile 项目中的几个核心线索。

## Soft control

Control often appears as smooth guidance rather than direct prohibition.

控制常以顺滑引导的方式出现，而不是直接禁令。

## Screen-switching and attention routing

Screen changes and rotating viewpoints can steer where attention goes.

屏幕切换与视角旋转会持续导引注意力。

## Prediction and updating

Continuous prediction and updating can turn choice into routine response.

持续预测与更新会把“选择”转化为“惯性反应”。

## Game engine as performance infrastructure

The game engine is both medium and infrastructure for image, timing, interaction, and feedback.

游戏引擎既是媒介，也是组织图像、时间、互动与反馈的基础设施。
